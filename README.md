## Install

* **Test this configuration with:**

```bash

git clone https://github.com/nvim-lua/config.nvim.git "${XDG_CONFIG_HOME:-$HOME/.config}"/config.nvim

```

```bash

alias nvim-luz='NVIM_APPNAME="config.nvim" nvim'

```

```bash

nvim-luz

```
